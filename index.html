📝 gdocs-comments-mcp - Precise Google Docs AI Commenting

Download Version 1.0

💡 What this tool does

Google Docs lacks tools for AI agents to place precise, range-anchored comments. This software solves that problem. It acts as a bridge, known as an MCP server. It allows AI tools like Claude, Cursor, and Copilot to highlight specific parts of your documents and leave comments exactly where they belong.

This allows your AI assistants to provide feedback that connects to specific paragraphs, sentences, or phrases. You no longer need to translate AI advice manually. The agent does the work directly in your document.

⚙️ System Requirements

  • - Windows 10 or Windows 11
  • - A stable internet connection
  • - Google Chrome or Microsoft Edge installed
  • - An active Google account with access to the documents you wish to edit

📥 Download and Setup

Follow these steps to prepare the tool for your AI agent.

  1. Visit the official releases page on GitHub.
  2. Select the latest version available under the "Assets" section.
  3. Download the Windows installer file ending in .exe.
  4. Locate the file in your Downloads folder and double-click it to run the installer.
  5. Follow the on-screen instructions to complete the installation process.
  6. Restart your AI editor, such as Cursor or Claude Code, to recognize the new tool.

🚀 Using your new tool

Once you install the software, you must link it to your AI agent. Open your AI agent settings window. Locate the "MCP Servers" section. Add the path to the installed gdocs-comments-mcp file.

The agent will now see a new capability. When you ask the agent to review a Google Doc, it checks for this tool. It then gains the ability to create comments that anchor to specific text ranges. You will see these appear as standard, high-quality comments within your Google Doc interface.

🛡️ Privacy and Security

We built this tool with security in mind. It uses standard browser automation to interact with your documents. It does not store your Google login details. It performs actions only when you request them through your AI agent. Ensure you keep your software updated to receive the latest security patches.
